# TracePoint Pilot Data Bridge 2

This package connects the saved Range & Training workspace to officer-performance analytics and training alerts.

## What it adds

- `/api/pilot/performance-summary`
- Analytics page now reads saved pilot data from Supabase.
- Qualification trends are calculated from saved qualification-style drill results.
- Drill performance trends are calculated from saved drill results.
- Training Alerts inbox imports generated alerts from saved performance concerns.
- Existing local remediation workflow remains intact.

## Pilot meaning

Bridge 1 proved the Range & Training workspace saves to Supabase.

Bridge 2 uses that saved data so Analytics and Training Alerts are no longer purely static/demo content.
