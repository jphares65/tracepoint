# Settings Export Tools

This package makes the Settings → Import / Export area functional on both sides.

## Adds to `/settings/import-export`

- Personnel CSV export
- Firearms CSV export
- Ammunition CSV export
- Firearms import template download
- Personnel import template download
- Qualification history import template download

## Uses existing routes

- `/api/pilot/personnel`
- `/api/armory/firearms`
- `/api/pilot/ammunition`

No SQL is required.
